# Password Reset

This endpoint initiates the password set process.

---

```json
{
  "Host": "api.bringyour.com",
  "Content-Type": "application/json"
}
```

**Body:**

```json
{
  "reset_code": "string",
  "password": "string"
}
```

| Field | Type | Description |
| :--- | :--- | :--- |
| `reset_code` | `string` | **Required.** The unique verification code received via email or SMS. |
| `password` | `string` | **Required.** Your new account password. |

### Response

#### `200 Success`

**Status:** `200 OK`

**Content-Type:** `application/json`

```json
{}
```

A successful response returns an empty object, indicating the password has been successfully updated.

### Example

#### HTTP Request

```http
POST /auth/password-set HTTP/1.1
Host: api.bringyour.com
Content-Type: application/json

{
  "reset_code": "RST-1234-5678",
  "password": "NewSecurePassword!"
}
```

#### Successful HTTP Response

```json
{}
```

# Password Set

This endpoint allows you to establish a new password for your account after initiating a password reset. You must provide the `reset_code` sent to your registered email or phone number.

---

## `POST /auth/password-set`

Submit a new password using a valid reset code.

### Request

`POST /auth/password-set`

**Headers:**

```json
{
  "Host": "api.bringyour.com",
  "Content-Type": "application/json"
}
```

**Body:**

```json
{
  "reset_code": "string",
  "password": "string"
}
```

| Field | Type | Description |
| :--- | :--- | :--- |
| `reset_code` | `string` | **Required.** The unique verification code received via email or SMS. |
| `password` | `string` | **Required.** Your new account password. |

### Response

#### `200 Success`

**Status:** `200 OK`

**Content-Type:** `application/json`

```json
{}
```

A successful response returns an empty object, indicating the password has been successfully updated.

### Example

#### HTTP Request

```http
POST /auth/password-set HTTP/1.1
Host: api.bringyour.com
Content-Type: application/json

{
  "reset_code": "RST-1234-5678",
  "password": "NewSecurePassword!"
}
```

#### Successful HTTP Response

```json
{}
```
