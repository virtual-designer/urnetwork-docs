# Network Reliability

This endpoint provides detailed statistics regarding the global network's reliability, analyzed through time-series windows and geographical multipliers. It is used to understand the current stability and weighting of the total network infrastructure.

---

## `GET /network/reliability`

Retrieve the current reliability window and associated metrics.

### Request

`GET /network/reliability`

**Headers:**

```json
{
  "Host": "api.bringyour.com",
  "Authorization": "Bearer YOUR_JWT_TOKEN",
  "Accept": "application/json"
}
```

#### Fields

| Field | Type | Description |
| :--- | :--- | :--- |
| `reliability_window` | `object` | The time-window based reliability calculation. |
| `reliability_window.mean_reliability_weight` | `number` | The average reliability score across the network. |
| `reliability_window.reliability_weights` | `array` | Time-series of reliability scores (weights). |
| `reliability_window.client_counts` | `array` | Number of active clients per time bucket. |
| `reliability_window.country_multipliers` | `array` | Reliability weight factors applied per country based on local network conditions. |
| `error` | `object` | Optional error details. |
